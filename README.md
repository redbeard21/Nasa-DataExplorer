# NASA Data Explorer

Un dashboard interactivo en **Python** que consume datos reales de las **APIs públicas de la NASA**, mostrando imágenes astronómicas, fotografías del rover *Curiosity* en Marte y registros de asteroides cercanos a la Tierra.

Desarrollado en **Google Colab**, este proyecto combina **ciencia de datos**, **astronomía** y **visualización interactiva**.

---

## Descripción general

**NASA Data Explorer** es una herramienta educativa y exploratoria que conecta con tres fuentes oficiales de datos de la NASA:

1. **Astronomy Picture of the Day (APOD)** — Imagen astronómica diaria con título y descripción.  
2. **Mars Rover Photos API** — Fotografías del rover *Curiosity* en Marte, filtradas por fecha.  
3. **Near Earth Object Web Service (NeoWs)** — Datos de asteroides cercanos a la Tierra, con representación gráfica.

El objetivo es demostrar cómo integrar **APIs científicas**, procesar datos reales y visualizarlos de forma sencilla y accesible.

---

## Tecnologías utilizadas

- **Python 3.12**
- **Google Colab**
- **Requests** → para conexión con APIs
- **Pandas** → para estructurar los datos
- **Matplotlib** → para visualización gráfica
- **JSON** → para analizar las respuestas de la NASA API
